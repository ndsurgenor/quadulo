Rules

Goal

Click as many blocks as you can before running out of possible moves

Gameplay

Blocks must be clicked in the repeated order 1-2-3-4; blocks > 4 cannot be selected. All blocks in the same row and column will increase by the value of the selected block. Selecting a block makes it unavailable until a 1 is clicked again.

Limits

Blocks will only ever increase as high as the current limit (initially set at 5). If a block would increase beyond the limit its value rolls back round to 1, 2, 3 etc. The limit will increase by 1 (max 9) for every level progressed i.e. every 16 blocks cleared.
